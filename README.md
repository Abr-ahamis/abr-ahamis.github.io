# Neo-virex Portfolio

A cybersecurity portfolio website for Abraham Eshetu (Neo-virex), showcasing projects, coursework, and blog posts.

## Features

- Terminal-themed design
- Responsive layout
- Project showcases with GitHub links
- Blog section
- Contact information

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome icons

## Local Development

1. Clone the repository
2. Open `index.html` in your browser or run a local server

## Deployment

This site is deployed on GitHub Pages.
